Tarball/Extracted Programs

Programs distributed as .tar.gz or .tar.xz archives.

Setting Up an Extracted Program

# Extract the archive
tar -xzf program.tar.gz -C ~/Applications/

# Or for .tar.xz
tar -xJf program.tar.xz -C ~/Applications/

# Find the executable
ls ~/Applications/program/
# Often in bin/ directory: ~/Applications/program/bin/program

Example Desktop Entries

JetBrains IDE (PyCharm, IntelliJ, etc.)

~/.local/share/applications/pycharm.desktop 
[Desktop Entry]
Version=1.0
Type=Application
Name=PyCharm Community
GenericName=Python IDE
Comment=Python IDE for Professional Developers
Exec=/home/sarah/Applications/pycharm-community/bin/pycharm.sh %f
Icon=/home/sarah/Applications/pycharm-community/bin/pycharm.png
Categories=Development;IDE;
Terminal=false
StartupWMClass=jetbrains-pycharm-ce

Programs Needing a Working Directory

Some programs need to run from their installation directory. Use the Path field:

~/.local/share/applications/legacy-app.desktop 
[Desktop Entry]
Version=1.0
Type=Application
Name=Some Legacy App
Exec=/home/sarah/Applications/legacyapp/run.sh
Path=/home/sarah/Applications/legacyapp
Icon=application-x-executable
Categories=Utility;
Terminal=false
Finding Icons in Extracted Programs

Check these common locations:

  • ./icons/
  • ./share/icons/
  • ./resources/
  • ./bin/ (sometimes)

Or search: find ~/Applications/myapp -name "*.png" -o -name "*.svg" | head -20
