Icon Options

You have three options for the Icon field:

Option 1: Standard Icon Names

Use a name from your system's icon theme (no path, no extension):

Icon=firefox
Icon=utilities-terminal
Icon=application-x-executable

Option 2: Full Path to Image

Point directly to any PNG or SVG file:

Icon=/home/sarah/Applications/myapp/icon.png
Icon=/home/sarah/icons/custom-icon.svg

Option 3: Install Icon to Theme

Copy your icon to the hicolor theme for cleaner references:

# For PNG icons (use appropriate size directory)
mkdir -p ~/.local/share/icons/hicolor/256x256/apps
cp myapp-icon.png ~/.local/share/icons/hicolor/256x256/apps/myapp.png

# For SVG icons (scalable)
mkdir -p ~/.local/share/icons/hicolor/scalable/apps
cp myapp-icon.svg ~/.local/share/icons/hicolor/scalable/apps/myapp.svg

# Update icon cache
gtk-update-icon-cache ~/.local/share/icons/hicolor/ 2>/dev/null || true

Then use just the name:

Icon=myapp

Custom Icons

Downloading Icons

You can find free icons at:

  • icon-icons.com
  • iconarchive.com
  • flaticon.com

Converting Icon Formats

# ICO to PNG
convert icon.ico icon.png

# SVG to PNG (specific size)
convert -background none icon.svg -resize 256x256 icon.png

# Resize PNG
convert icon.png -resize 256x256 icon-256.png

Creating Icons from Scratch

If your program has no icon, you can create a simple one with ImageMagick:

# Create a simple colored square
convert -size 256x256 xc:#4a90d9 ~/icons/myapp.png

# With text/initials
convert -size 256x256 xc:#4a90d9 -fill white -gravity center \
    -pointsize 72 -annotate 0 "MA" ~/icons/myapp.png
