Installation

Installing and authenticating Claude Code on Linux.

Method 1: Native Installer (Recommended)

This is the official recommended method. No Node.js required.

# Download and run the installer
curl -fsSL https://claude.ai/install.sh | bash

# Reload your shell configuration
source ~/.bashrc    # or source ~/.zshrc for zsh users

# Verify the installation
claude --version
claude doctor

The installer places the binary at ~/.local/bin/claude or ~/.claude/bin/claude.

Installing a Specific Version

# Install specific version
curl -fsSL https://claude.ai/install.sh | bash -s -- --version 2.0.30

# Install from stable channel (typically one week behind latest)
curl -fsSL https://claude.ai/install.sh | bash -s stable

Method 2: npm Installation (Deprecated)

Still works but migration to native installer is recommended.

# First, set up a user npm directory to avoid permission issues
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Verify
claude --version

Migrating from npm to Native

If you have an existing npm installation:

claude install

Your configuration files will be preserved during migration.

Alpine Linux Setup

Alpine and other musl-based distributions need additional dependencies:

apk add libgcc libstdc++ ripgrep
export USE_BUILTIN_RIPGREP=0
curl -fsSL https://claude.ai/install.sh | bash

Authentication

Claude Code requires a paid account. You have several options:

Option 1: Claude Pro or Max Plan (Recommended)

Subscribe at claude.ai. Provides unified billing for both Claude Code and Claude on the web.

# Launch Claude Code and follow OAuth prompts
claude

Option 2: Anthropic API Console

Pay-per-use via API credits. Can get expensive with heavy usage.

# Set API key as environment variable
export ANTHROPIC_API_KEY="sk-ant-your-key-here"

# Or add to your shell profile for persistence
echo 'export ANTHROPIC_API_KEY="sk-ant-your-key-here"' >> ~/.bashrc
source ~/.bashrc

Verifying Authentication

claude doctor

This command checks your installation and authentication status.

Tip

Your credentials are stored locally in ~/.claude/. If you need to re-authenticate or switch accounts, you can remove this directory and run claude again.
