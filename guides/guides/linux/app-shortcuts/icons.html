Choosing an Icon

How to use standard system icons or custom icon files for your app shortcuts.

Contents
  • Icon Types
  • Standard Icon Names
  • Custom Icons
  • Finding Available Icons

Icon Types

The Icon field accepts either:

  • A standard icon name (without file extension) that exists in your icon theme
  • An absolute path to an image file (PNG, SVG, or XPM)

Using Standard Icon Names

Standard icons are found in /usr/share/icons/ and follow the freedesktop.org naming specification. Just use the name without the path or extension:

Icon=text-html
Icon=applications-internet
Icon=utilities-terminal

Using a Custom Icon File

Point to any PNG or SVG file with a full path:

Icon=/home/sarah/icons/my-custom-icon.png
Icon=/home/sarah/Projects/dashboard/favicon.svg

Custom Icons

Option 1: Direct Path (Simplest)

Just point to any image file:

Icon=/home/sarah/icons/my-app-icon.png

Supported formats: PNG (recommended), SVG, XPM

Option 2: Install to Icon Theme (Cleaner)

For a more "proper" installation, copy your icon to the hicolor theme:

# For a 48x48 pixel PNG
mkdir -p ~/.local/share/icons/hicolor/48x48/apps
cp my-icon.png ~/.local/share/icons/hicolor/48x48/apps/my-app-name.png

# For an SVG (scalable)
mkdir -p ~/.local/share/icons/hicolor/scalable/apps
cp my-icon.svg ~/.local/share/icons/hicolor/scalable/apps/my-app-name.svg

# Update icon cache
gtk-update-icon-cache ~/.local/share/icons/hicolor/ 2>/dev/null || true

Then in your .desktop file, just use the name without extension:

Icon=my-app-name

Extracting Icons from Websites

If your HTML app has a favicon, you can use it:

# Download a favicon
wget https://example.com/favicon.ico -O ~/icons/my-app.ico

# Convert ICO to PNG (if needed)
convert ~/icons/my-app.ico ~/icons/my-app.png

Finding Available Icons

To see what icons are available on your system:

# List icons in the default theme
ls /usr/share/icons/hicolor/48x48/apps/
ls /usr/share/icons/Adwaita/48x48/apps/

# Search for a specific icon
find /usr/share/icons -name "*web*" -type f 2>/dev/null | head -20

# List all standard apps icons
ls /usr/share/icons/hicolor/scalable/apps/
