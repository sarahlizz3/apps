Quickemu Windows VM

Setting up a Windows 11 VM as an "Apple TV appliance" using Quickemu.

1. Initial Setup

Install Quickemu and download Windows 11:

sudo apt-add-repository ppa:flexiondotorg/quickemu
sudo apt update
sudo apt install quickemu
quickget windows 11

2. DRM & Hardware Spoofing

Edit your windows-11.conf and add these for the EliteBook 840 G7:

cpu_model="host,hv_vendor_id=genuineintel"
arg="-smbios type=1,manufacturer='HP',product='EliteBook 840 G7'"
ram="4G"

3. Portability

To move to the TV Laptop (since usernames and folders match):

  • Copy the .conf and the windows-11/ folder to ~/VMs on the G7
  • Run: quickemu --vm windows-11.conf --shortcut

4. Full Removal

To wipe the VM and reclaim 20GB+ of space:

rm -rf ~/VMs/windows-11*
rm ~/.local/share/applications/windows-11.desktop
