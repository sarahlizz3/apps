Draft Diff Checker

A local browser tool for comparing student draft revisions. All processing happens on your computer - no data is ever sent anywhere.

Quick Reference
Task How To
Open the tool Open index.html in your browser
Compare drafts Drag files to upload zones, click "Compare Drafts"
Adjust sensitivity Use the 30%/50%/70% threshold buttons

First-Time Setup

Before using the tool, you need to download three library files. This only needs to be done once.

  1. Create a lib folder

    Create a folder called lib in the same directory as index.html

  2. Download the library files

    Download these three files and save them in the lib folder:

    PDF.js (main library):

    https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js

    Save as: lib/pdf.min.js

    PDF.js (worker):

    https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.worker.min.js

    Save as: lib/pdf.worker.min.js

    Mammoth.js (Word document parser):

    https://cdnjs.cloudflare.com/ajax/libs/mammoth/1.6.0/mammoth.browser.min.js

    Save as: lib/mammoth.browser.min.js

  3. Verify folder structure

    Your folder should look like this:

    diff-checker/
├── index.html
├── style.css
├── diff.js
├── README.txt
└── lib/
    ├── pdf.min.js
    ├── pdf.worker.min.js
    └── mammoth.browser.min.js

Using the Tool

  1. Open index.html in your web browser (Chrome, Firefox, Edge, Safari)
  2. Drag the original draft onto the left upload zone
  3. Drag the revised draft onto the right upload zone
  4. Click "Compare Drafts"
  5. Use the threshold buttons (30%/50%/70%) to adjust how changes are displayed

What the Colors Mean

Color/Style Meaning
Red strikethrough Text that was deleted
Dark green text Text that was added
Yellow highlight Section that was heavily revised (shown as a chunk)
Green ¶ Paragraph break was added
¶ Paragraph break was removed
Privacy

All processing happens in your browser using JavaScript. The library files (PDF.js and Mammoth.js) are open-source tools from Mozilla and the mammoth project. Once downloaded, this tool works completely offline with zero network access required. No data is ever sent anywhere - your files stay on your computer.
