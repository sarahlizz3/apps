Daily Use

Quick reference for everyday tasks with Stremio Recommender

Starting the Addon

The addon needs to be running for Stremio to show recommendations.

  1. Open a terminal

    Use your preferred terminal application.

  2. Navigate to the addon folder
    cd ~/Apps/stremio-recommender
  3. Start the addon
    npm start

    You should see output indicating the server is running on port 7000.

Tip

Want the addon to start automatically? See the Background Service section in Troubleshooting.

Using in Stremio

First Time Setup

If this is your first time using the addon, you need to install it in Stremio:

  1. Make sure the addon is running
  2. Open Stremio
  3. Go to Settings > Addons
  4. Click "Add Addon" and enter: http://localhost:7000/manifest.json
  5. Click Install

Finding Recommendations

  1. Open Stremio
  2. Go to the Discover tab
  3. Look for "AI Recommendations" in the catalog dropdown
  4. Browse your personalized picks
Note

If you don't see any recommendations, you may need to generate them first using the dashboard.
