Dashboard

The web interface for managing your recommendations

Opening the Dashboard

  1. Make sure the addon is running (npm start)
  2. Open your browser
  3. Go to http://localhost:7000

Dashboard Features

Status Overview

The top of the dashboard shows:

  • Addon Status: Whether the addon is running and connected
  • Last Generated: When recommendations were last generated
  • Total Recommendations: How many titles are in your current list

Current Recommendations

A list of your current recommendations with:

  • Title and year
  • Match score (how well it fits your taste)
  • Brief reason for the recommendation
  • Dismiss button to remove unwanted titles

Dashboard Buttons

Button What It Does
Generate Fetches new titles from TMDb and scores them with AI. This can take a few minutes.
Refresh Reloads the recommendation list without generating new ones.
Clear All Removes all current recommendations. You'll need to generate new ones.
Edit Taste Profile Opens an editor for your taste preferences.
Heads Up

Generating recommendations uses your Claude API credits. Each generation typically processes 50-100 titles.

Understanding Scores

Each recommendation has a match score from 1-10:

  • 8-10: Excellent match - highly likely to enjoy
  • 6-7: Good match - worth checking out
  • 4-5: Moderate match - might enjoy
  • 1-3: Weak match - probably not your thing

By default, only titles scoring 6 or higher are shown in Stremio.
