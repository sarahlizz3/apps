Managing Recommendations

How to dismiss, refresh, and customize your recommendations

Dismissing Titles

When you've seen a title or know you're not interested, dismiss it so it won't appear again.

Via Dashboard

  1. Open the dashboard at http://localhost:7000
  2. Find the title you want to dismiss
  3. Click the "Dismiss" button next to it

Dismissed titles are stored in a file and won't be recommended again, even after generating new recommendations.

Via File (Bulk Dismiss)

You can also edit the dismissed titles file directly:

~/Apps/stremio-recommender/data/dismissed.json

The file contains an array of TMDb IDs:

[
  { "id": 123456, "title": "Movie Title", "dismissedAt": "2024-01-15" },
  { "id": 789012, "title": "Another Movie", "dismissedAt": "2024-01-16" }
]
Tip

You can find a movie's TMDb ID by searching for it on themoviedb.org - it's in the URL.

Refreshing Recommendations

There are two ways to get fresh recommendations:

Generate New

Click "Generate" in the dashboard to:

  • Fetch the latest trending and popular titles from TMDb
  • Score them against your taste profile
  • Replace your current recommendations with new ones

This uses API credits but gives you completely fresh suggestions.

Refresh Display

Click "Refresh" to reload the current recommendations list. Use this if:

  • You dismissed some titles and want to see the updated list
  • The display seems out of sync

Updating Your Taste Profile

Your taste profile determines what gets recommended. Edit it to improve suggestions:

Location

~/Apps/stremio-recommender/taste-profile.json

What to Include

  • Favorite genres: Action, sci-fi, comedy, etc.
  • Preferred themes: Time travel, heist, found family, etc.
  • Favorite actors/directors: People whose work you consistently enjoy
  • Dislikes: Things you want to avoid (horror, excessive gore, etc.)
  • Examples: Specific movies/shows you love as reference points

Example Profile

{
  "likes": {
    "genres": ["sci-fi", "thriller", "dark comedy"],
    "themes": ["time travel", "heist", "found family", "redemption arcs"],
    "elements": ["clever dialogue", "plot twists", "ensemble casts"]
  },
  "dislikes": {
    "genres": ["horror", "slasher"],
    "elements": ["jump scares", "excessive gore", "animal death"]
  },
  "examples": {
    "loved": ["Inception", "The Grand Budapest Hotel", "Severance"],
    "disliked": ["Saw", "The Human Centipede"]
  }
}
After Updating

Generate new recommendations after updating your taste profile to see the changes take effect.
