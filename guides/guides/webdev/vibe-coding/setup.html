Setup

Prerequisites, installation, and authentication for Claude Code.

Prerequisites

Node.js (version 18 or higher)

Claude Code requires Node.js to run. Check your version:

node --version
macOS

If you need to install or update Node.js:

# Using Homebrew (recommended)
brew install node

# Or download directly from nodejs.org
Pop!_OS / Linux

The default Node.js in Ubuntu/Pop!_OS repos is often outdated. Use NodeSource:

# Add NodeSource repository for Node 20.x
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# Install Node.js
sudo apt install -y nodejs

# Verify installation
node --version

VS Code

Any recent version works. Claude Code runs in the integrated terminal, so no special extension is required.

Installation

Install Claude Code globally via npm:

npm install -g @anthropic-ai/claude-code
macOS

If you get permission errors, fix npm's global directory permissions or use a Node version manager like nvm:

# Option 1: Fix permissions
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.zshrc
source ~/.zshrc

# Then install again
npm install -g @anthropic-ai/claude-code
Pop!_OS / Linux

Avoid using sudo with npm global installs. Instead, configure npm to use a directory in your home folder:

# Create directory for global packages
mkdir -p ~/.npm-global

# Configure npm to use it
npm config set prefix '~/.npm-global'

# Add to your shell (bash)
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# Or if using zsh
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.zshrc
source ~/.zshrc

# Now install
npm install -g @anthropic-ai/claude-code

Verify the installation:

claude --version

Authentication

Before using Claude Code, you need to authenticate. Run the following command and follow the prompts:

claude

You'll be given options to:

  • Log in with your Anthropic account — Opens a browser for OAuth authentication
  • Use an API key — Enter your Anthropic API key directly
Pop!_OS / Linux

If the browser doesn't open automatically for OAuth, the terminal will display a URL. Copy and paste it into your browser manually.

API Key Storage

Your credentials are stored locally in ~/.claude/. If you need to re-authenticate or switch accounts, remove this directory and run claude again.
